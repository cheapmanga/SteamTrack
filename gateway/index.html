steamtrack

Stable entry point. The site runs behind a tunnel whose address changes; this page always points at the current one.

steamtrack follows a hand-picked list of Steam games and records what changes about them — new builds, branches and depots, store page edits, price moves, patch notes, player counts. It reads Steam’s own public PICS feed as an anonymous client, the same source SteamDB uses, and exposes everything through a browsable site and a public API.
